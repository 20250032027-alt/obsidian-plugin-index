# ObsidianPluginIndex

Community-ranked Obsidian plugins, built from real r/ObsidianMD data.

995 posts. 2,824 comments. Zero sponsored slots.

## Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Vercel will auto-detect the static site. No build config needed.
4. Deploy.

## What's inside

- `index.html` — the full site, self-contained
- `vercel.json` — Vercel routing config
- `src/data.js` — raw plugin data (for reference; data is also embedded in index.html)

## Data source

All plugin rankings come from r/ObsidianMD post upvotes. The database contains scraped posts and comments with no editorial filtering applied. A plugin is listed because the community upvoted it — not because anyone asked to be included.

## Stack

Plain HTML, CSS, and vanilla JS. No framework, no build step, no dependencies to maintain. Fast by default.
